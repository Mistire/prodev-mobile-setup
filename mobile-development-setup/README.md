# 0. Setting Up and Testing Your Mobile Development Environment

## Objective
Set up a mobile development environment using **Expo Framework for React Native** to test applications on a physical device with **Expo Go**.

## Prerequisites
- Node.js LTS
- VS Code
- macOS, Linux, or Windows
- Expo Go installed on a mobile device (Android or iOS)

## Installing Expo Go
1. Visit [Expo Go](https://expo.dev/go) and select the latest SDK version.  
2. Install Expo Go on your device:  
   - **Android:** Google Play Store  
   - **iOS:** Apple App Store  
3. Open Expo Go and log in or create an account.

## Verifying the Setup
```bash
npx create-expo-app my-first-app
cd my-first-app
npx expo start
